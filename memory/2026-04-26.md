## PrivaShield – Audit/PDCA-Verzahnung & erweiterter Mandanten-/Gruppenexport / Release 1.6.5
- Echte fachliche Referenzierung zwischen Audits und PDCA ergänzt:
  - `audits.verknuepftePdcaIds` als explizite Liste verknüpfter PDCA-Zyklen
  - `pdca.verknuepftesAuditId` als Rückverweis vom PDCA-Zyklus auf ein Audit
- Audit-UI erweitert:
  - Audit-Formular kann jetzt verknüpfte PDCA-Zyklen pflegen
  - Audit-Seite wertet explizite Verknüpfungen bevorzugt aus und zeigt zugeordnete PDCA-Follow-ups direkt im Audit an
  - Kennzahlenblock für offene Audit-Follow-ups / Verzahnung beibehalten
- PDCA-UI erweitert:
  - PDCA-Formular kann jetzt direkt mit einem Audit verknüpft werden
- Export / Druck erweitert:
  - Audit-/PDCA-Managementblock zeigt jetzt auch explizite Audit↔PDCA-Verknüpfungen
  - Mandantenexport enthält zusätzlich eine ausgebaute Gruppen-/Verbundsicht mit Governance-, Rollen-, Risiko-, Reifegrad-, Ampel- und Roadmap-Blöcken für gruppierte Mandanten
- Version gemäß Patch-Schema auf `1.6.5` erhöht.
- TypeScript-Check (`npm run check`) erfolgreich.
- Tests (`npm test`) erfolgreich, 29/29 Tests grün.
- Commit `c6647d8` erstellt (`feat: link audits with pdca and expand tenant export`).
- Änderungen erfolgreich nach `master` gepusht; zusätzlicher Versions-Branch `1.6.5` erstellt und gepusht.
